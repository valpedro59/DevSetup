Pour utiliser un repository il faut suivre les etapes suivantes :

1. Installer Git et configurer ses id

2. Creer un repository en tapant la commande "git init"

3. Enregistrer ses fichiers par "git add. "

4. Valider ses modifications grace au commit par "git commit -m''. "

3. Envoyer le depot en ligne "git push origin branch. "
